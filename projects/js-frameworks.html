About the project

This project involved building a complete eCommerce storefront using React and the Noroff Online Shop API. The application includes essential eCom features such as product listings, dynamic product pages, cart management, a checkout flow and a validated contact form. I used React Router for navigation, designed reusable components and implemented global state for the cart. The homepage features a look-ahead search bar, improving usability and product discovery. The checkout success page clears the cart and guides the user back to the store, creating a full end-to-end shopping experience. The project is fully responsive and deployed on Netlify, showcasing my ability to structure a React application, manage data flows and build interactive, user-friendly interfaces from scratch.

Tech stack

  • React + TypeScript: Core logic, components, and strong typing for safer code
  • React Router DOM: SPA navigation and dynamic product routes
  • Context API + Reducer: Global cart state management and predictable updates
  • CSS Modules: Scoped, maintainable styling for each component
  • Vite: Fast development environment and optimized production builds
  • Noroff API: Product data, pricing, discounts, and reviews
