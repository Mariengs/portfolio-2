About the project

The purpose of this project was to design and develop a fully functional front-end auction platform that interacts with a real-world API. The application is aimed at both registered users who want to create listings and place bids, and non-registered users who want to browse available auctions. The main features include user registration and authentication, profile management, creation and administration of auction listings, and a bidding system with bid history and countdown timers. I focused on creating a clean, intuitive, and responsive user experience that feels reliable and easy to navigate. Accessibility, clear feedback, and a structured layout were important priorities to ensure the application works well across different devices and user needs.

Tech stack

  • HTML
  • CSS / Tailwind CSS
  • JavaScript (Vanilla)
  • External API
